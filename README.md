# 📊 SCT_DA_4 - Business Insights Report (EDA)

## 📌 Project Overview

This project was completed as part of the **SkillCraft Technology Data Analytics Internship**. The goal was to perform Exploratory Data Analysis (EDA) on a marketing campaign dataset and generate actionable business insights through data visualization.

## 🎯 Objectives

* Clean and prepare the dataset
* Analyze customer demographics and spending behavior
* Evaluate marketing campaign performance
* Identify high-performing sales channels
* Generate business recommendations based on data insights

## 🛠 Tools & Libraries

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## 📈 Analysis Performed

* Missing Value Handling
* Duplicate Removal
* Data Type Conversion
* Customer Age Distribution Analysis
* Income vs Total Spending Analysis
* Purchase Channel Analysis
* Campaign Response Analysis
* Product Category Spending Analysis

## 🔍 Key Insights

* Higher-income customers generally spend more.
* Store purchases contribute the highest number of transactions.
* Campaign response rate is relatively low, indicating opportunities for marketing optimization.
* Customer spending varies significantly across product categories.
* Customer demographics influence purchasing behavior.
